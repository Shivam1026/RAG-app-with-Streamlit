# 📘 Question Answering RAG System

A Retrieval-Augmented Generation (RAG) system for question answering using:
- **Frontend**: Streamlit
- **Backend**: Python
- **Model**: Open-source Hugging Face transformer

---

## 🚀 Features
- Interactive **Streamlit UI** for user queries
- **Document ingestion** and embedding storage
- **Retriever** to fetch relevant context
- **Hugging Face model** for answer generation
- Modular design for scalability

---

## 🛠️ Tech Stack
| Component   | Technology |
|-------------|------------|
| Frontend    | Streamlit  |
| Backend     | Python     |
| Embeddings  | Hugging Face / SentenceTransformers |
| Vector Store| FAISS / Chroma |
| Model       | Hugging Face Transformers |
